# Prelude

みなさま、こんにちは。
ある日突然、Servo や Firefox などの Mozilla なプロジェクトにコントリビュートしたい！って思い立つ時ってありますよね？

そんな時にすぐにコントリビュート開始できるように、読むべきドキュメントを独断と偏見でまとめました。

（主に自分向けです）
（間違いがあったり、これも追加したほうがいいよねというものが合った場合は、`編集リクエスト`でお知らせください）

# Servo

- [Servo 本家](https://servo.org/)
- [Servo Blog](https://blog.servo.org/)
- [Servo Starters](https://starters.servo.org/)
- [Servo GitHub リポジトリ](https://github.com/servo/servo)

`Servo本家`は Servo プロジェクトのサイトですね。
Servo とはなんぞや？を知りたいときは、ここを見るのが早いかも。

`Servo Starters`は、Servo 初心者がコントリビュートを始めたいけど、どこからはじめたらいいんだっけ？に答えてくれるサイト。
言語別（Rust, Python, JS...）に初心者向けの Pull Request がリストアップされていてここを漠然と眺めているだけでも、面白いです。
（まさに今の私にうってつけ！）

`Servo Blog` は一週間ごとのプロジェクトの動きが分かります。

`Servo GihHubリポジトリ`は、そのままです。

以下の記事も（ちょっと古いですが）Servo プロジェクトがドコを目指しているのかがパッと見てわかってよかったです。

- [Servo の現況概説的なおはなし](http://saneyukis.hatenablog.com/entry/2013/07/16/004928) (ちょっぴり古い)

## Rust (主に日本語)

せっかく Servo のコントリビューターを目指すんなら、Rust も読み書きできるようにならないとね！

- [プログラミング言語 Rust](http://rust-lang-ja.github.io/the-rust-programming-language-ja/1.6/book/)
- [Rust by Example](http://rust-lang-ja.org/rust-by-example/index.html)

Rust はドキュメントの翻訳プロジェクトが活発に活動してるみたいですね！
こういったところでもコントリビュートできそうですね！

- [Rust のドキュメントの翻訳プロジェクトを開始します。](http://qiita.com/blackenedgold/items/3b068769736e671805f0)
- [プログラミング言語 Rust の翻訳プロジェクト・リポジトリ](https://github.com/rust-lang-ja/the-rust-programming-language-ja)
