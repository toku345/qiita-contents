[showdown](https://rubygems.org/gems/showdown)という Gem を(ちょっと前に)作ったので、勝手に紹介させていただきます。

# showdown とは？

Ruby で作った Markdown ビューワーです。

![screencast.gif](https://qiita-image-store.s3.amazonaws.com/0/13998/b7d50137-7cfb-8488-64a9-c02be042bd83.gif)

ファイルの更新を検出して、（ほぼ）リアルタイムで変更が反映されます。

# インストール

※ Ruby が入っていない場合は、rbenv 等を使って Ruby をイントールしてください。

```
$ gem install showdown
```

# 使い方

```
$ showdown /path/to/your.md
tartget: /path/to/your.md
Thin web server (v1.6.3 codename Protein Powder)
Maximum connections set to 1024
Listening on 0.0.0.0:3000, CTRL+C to stop
```

ブラウザで `http://localhost:3000` にアクセス！
指定したファイルをエディタで更新すると、ブラウザ上のプレビューも更新されます！

詳しい GitHub の README を御覧ください。
https://github.com/toku345/showdown
